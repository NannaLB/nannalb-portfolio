Sweetbot website

About the project

The assignment was to create a Danish website for the company Sweetbot.Design. They had a styleguide that should be followed, and a logo to implement. Furthermore, sweetbot wanted an e-commerce called HotBot that should contain icons and a logo. These had to be developed and integrated on the site.

See full project description

Category

Website design, Illustrator, Photoshop, Html, CSS, jQuery

The result

We created a danish website that contained all of the demands from Sweetbot.Design. But after several usertests, a couple of changes had to be made. We needed to simplify the connection between Sweetbot and HotBot, because none of the users we tested knew how to find HotBot through the banner on the frontpage. We also had to minimize the pictures and the text on the sweetbot-site, because the usertest showed us that it was too confusing and overwhelming, with the original design.
